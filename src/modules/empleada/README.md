# Modulo empleada

Este modulo contiene todo lo relacionado con la experiencia del rol empleado/empleada.

Archivos principales:
- `pages/PanelEmpleadaPage.tsx`: panel principal con creacion y listado de tickets propios.
- `pages/DetalleTicketEmpleadaPage.tsx`: vista de detalle del ticket e historial de comentarios.
- `services/ticketsEmpleada.ts`: consultas de Supabase usadas por este rol.
- `constantes.ts`: prioridades y estilos del modulo.
- `utilidades.ts`: helpers del modulo.
